## 🔄 5개 탭 구성

### 1. **홈** 탭
- **주소** (펼치기 가능)
- **거리 정보** (걸어서 아이콘)
- **오늘 운영시간** (펼치기 → 7일 전체 표시)
- **전화번호** (복사 버튼)
- **진료과목 미리보기** (3개만, ... 표시)
- **정보 더보기** 버튼 (소개 탭으로 이동)

### 2. **운영시간** 탭
- 7일 시간표 (리스트형)
- 오늘 요일 핑크 배경 하이라이트
- 휴무 표시

### 3. **진료** 탭
- 진료과목 리스트
- Bullet point (핑크 dot)
- 각 항목 구분선

### 4. **시설** 탭
- 2x2 Grid
- 큰 아이콘 (64px)
- 배경색 구분

### 5. **소개** 탭
- 병원 설명 텍스트
- 읽기 편한 line height (24)

## 💡 UX 개선

### **정보 탐색**
- 탭 클릭 또는 좌우 스와이프
- 각 탭이 전체 화면 사용
- 명확한 정보 분리

### **인터랙션**
- 펼치기/접기 (주소, 운영시간)
- 복사 버튼 (전화번호)
- Haptic feedback 모든 액션
- "정보 더보기" → 소개 탭으로

### **정보 우선순위** (홈 탭)
1. 주소 (가장 중요)
2. 거리
3. 오늘 운영시간
4. 전화번호
5. 진료과목 미리보기

## 🧪 테스트 방법

1. **iOS 시뮬레이터** 새로고침
2. **병원 카드** 클릭
3. **헤더 확인**:
   - 병원명 + 알림받기
   - 평점 + 리뷰
   - 전화하기/길찾기 버튼
   - 저장/좋아요/공유 아이콘
4. **탭 테스트**:
   - 탭 클릭으로 전환
   - 좌우 스와이프로 전환
5. **홈 탭 확인**:
   - 주소 펼치기
   - 오늘 운영시간 펼치기
   - 전화번호 복사
   - 정보 더보기 버튼
