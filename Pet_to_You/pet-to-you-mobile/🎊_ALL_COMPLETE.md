## 🚀 실행 방법 (3가지)

### 방법 1: 자동 스크립트 (가장 쉬움) ⭐

```bash
./EXECUTE_NOW.sh
```

### 방법 2: 수동 실행 (2개 터미널)

**터미널 1 - 백엔드**:
```bash
cd ../pet-to-you-api
npm run start:dev
```

**터미널 2 - 모바일**:
```bash
npx expo start --clear
```

### 방법 3: 한 줄 명령어

```bash
(cd ../pet-to-you-api && npm run start:dev &); sleep 5; npx expo start --clear
```
