# Pet to You 🐾

> Smart Pet Care Ecosystem - 반려동물 의료, 돌봄, 입양 통합 플랫폼

**버전**: v2.0
**업데이트**: 2025-12-29

## 프로젝트 소개

Pet to You는 대한민국 반려동물 서비스 시장의 **정보 비대칭성과 신뢰도 문제**를 해결하는 통합 플랫폼입니다.

**4가지 핵심 서비스**:
- 🏥 **병원 예약**: 의료 서비스 접근성 개선
- 🏫 **유치원 예약**: 투명한 가격과 인증된 시설
- 🐕 **입양 플랫폼**: 신뢰할 수 있는 보호소 연결
- 💰 **펫 보험**: 상품 비교 + 맞춤 추천 + 간편 청구

## 주요 기능 (v2.0)

### 병원 서비스
- 🏥 **병원 검색 & 예약**: 위치, 진료시간, 전문분야 기반 실시간 예약
- 🐕 **반려동물 관리**: 다중 반려동물 프로필 및 건강 정보 관리
- 📋 **건강수첩**: 진료 기록, 예방접종, 투약 이력 통합 관리
- 🔔 **알림 시스템**: 예약 알림, 건강 체크업 리마인더

### 유치원 서비스 ⭐ NEW
- 🏫 **유치원 검색**: 위치 기반 검색, 가격 비교, 인증 필터
- 💰 **투명한 가격**: 시간제/종일제/월정액 가격표 공개
- ✅ **시설 인증**: OCR + 공공데이터 API 기반 자격 검증
- 📊 **활동 리포트**: 일일 활동 요약 (식사, 놀이, 낮잠)

### 입양 서비스 ⭐ NEW
- 🐕 **입양 동물 검색**: 인증된 보호소의 입양 가능 동물
- ✅ **보호소 신뢰도 검증**: 사업자 인증, 신고 이력 확인
- 🚫 **가짜 보호소 필터링**: 파양비 착취 방지
- 💬 **입양 신청 & 상담**: 온라인 신청 및 예약
- 📈 **투명한 정보**: 건강 기록, 성격 평가, 입양 성공률

### 펫 보험 서비스 ⭐ NEW
- 💰 **보험 상품 비교**: 5개 보험사 상품 한눈에 비교
- 🎯 **AI 맞춤 추천**: 반려동물 정보 기반 최적 상품 추천
- 📄 **간편 청구**: 진료 기록 연동 자동 서류 생성 (30분 → 3분)
- 📊 **청구 내역 추적**: 접수 → 심사 → 승인 → 지급 실시간 추적
- 🔗 **보험사 연결**: 정보 자동 입력으로 빠른 가입

## 기술 스택

### 프론트엔드
- **Framework**: React Native
- **상태 관리**: Redux Toolkit
- **네비게이션**: React Navigation
- **UI Library**: React Native Paper

### 백엔드
- **Framework**: Node.js + NestJS
- **Database**: PostgreSQL
- **ORM**: Prisma
- **캐싱**: Redis
- **인증**: JWT (Passport.js)

### 인프라
- **Cloud**: AWS
- **서버**: EC2/ECS
- **데이터베이스**: RDS (PostgreSQL)
- **파일 저장**: S3
- **CDN**: CloudFront

## 시장 규모 & 기회

- **TAM**: 14,000억원 (유치원 3,000억 + 입양 600억 + 보험 4,700억 + 병원)
- **성장률**: 연 15-30% (보험 30%, 유치원 15%, 입양 18%)
- **핵심 차별화**: 신뢰도 검증 + 정보 투명성 + 4-in-1 통합 플랫폼

## 프로젝트 타임라인

- **Phase 1 (12개월)**: MVP (병원 + 유치원 + 입양) - 138 SP
- **Phase 2 (6개월)**: 실시간 모니터링 + AI 매칭
- **Phase 3 (6개월)**: 커뮤니티 + 원격 진료 + 해외 진출
